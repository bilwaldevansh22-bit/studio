# **App Name**: TokenEstate

## Core Features:

- User Authentication: Enable users to log in via Google, email, phone number, or Facebook.
- NFT Creation: Allow users to create NFT representations of real estate properties.
- Fractional Sales: Facilitate the sale of fractional ownership of real estate through NFTs.
- MetaMask Integration: Integrate MetaMask wallet for secure crypto asset transactions.
- Real Estate Token Feed: Display a feed of real estate tokens on the home page.
- Token Search: Enable searching for real estate tokens with filtering options.

## Style Guidelines:

- Primary color: Desaturated blue (#64B5F6) to evoke trust and security.
- Background color: Very light desaturated blue (#E3F2FD) to provide a clean and calming backdrop.
- Accent color: A contrasting light purple (#9575CD) to draw attention to key interactive elements.
- Font pairing: Use 'Poppins' (sans-serif) for headings to give a fashionable feel, paired with 'PT Sans' (sans-serif) for body text.
- Employ minimalist and intuitive icons related to real estate and blockchain for clear user guidance.
- Incorporate subtle animations on property loading and transaction confirmations to enhance user engagement.
- Design a clear, intuitive layout with a focus on accessibility and optimal user experience.
